# 🌊 Sistema de Defeso – Plataforma Web Inteligente

Este projeto é uma aplicação web desenvolvida com o objetivo de auxiliar na gestão e análise de informações relacionadas ao período de defeso, oferecendo recursos tanto para usuários comuns quanto para administradores do sistema.

A plataforma permite o registro de dados, simulação de benefícios, envio de feedbacks e coleta de informações socioprofissionais, centralizando tudo em um ambiente organizado e automatizado.

---

## 🚀 Evolução do Projeto

O sistema foi inicialmente desenvolvido utilizando **PHP**, porém passou por uma **migração completa para Java com Spring Boot**.

Essa mudança não foi apenas de linguagem, mas de arquitetura e qualidade do sistema:

* Estruturação no padrão **MVC (Model-View-Controller)**
* Melhor organização e separação de responsabilidades
* Uso de **Spring Data JPA (Hibernate)** para persistência
* Implementação de **Spring Security** para controle de acesso
* Integração avançada com banco de dados (procedures, triggers e views)

---

## 🧠 Funcionalidades

### 🐟 Monitoramento de Fauna

* Registro de espécies marinhas
* Verificação automática de integridade dos dados
* Backup de exclusões via trigger
* Visualização otimizada com view no banco

---

### 💬 Central de Feedback

* Envio de mensagens pelos usuários
* Classificação automática (positivo, negativo, neutro)
* Execução de análise via procedure SQL
* Registro de histórico de exclusões

---

### 📊 Simulador de Benefício

* Simulação de elegibilidade para o seguro defeso
* Geração automática de resultado (aprovado/reprovado)
* Armazenamento do histórico por usuário
* Gerenciamento pelo painel administrativo

---

### 📚 Pesquisa Socioprofissional

* Coleta de dados sobre a atividade pesqueira
* Informações como tipo de embarcação, frequência e satisfação
* Base para análise estatística e tomada de decisão
* Visualização e controle pelo administrador
